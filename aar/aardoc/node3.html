Conventions

Names of files and folders as well as MATLAB commands are typeset in a typewriter font. Any word typeset in typewriter font and starting with the symbol $ represents a non-literal string, i.e. a string whose value might be different for different operating systems, MATLAB versions, etc. For instance, $MATLABROOT denotes the folder where MATLAB is installed.
